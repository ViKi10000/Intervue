# 💼 Intervue – Interview Platform

A full-stack interview platform inspired by modern hiring solutions.  
This project enables companies to conduct technical interviews, manage candidates, and streamline the hiring workflow.

---

## 🚀 Overview

Intervue is designed to simulate a real-world **technical interview platform**, providing tools for scheduling interviews, evaluating candidates, and managing the hiring process efficiently.

---

## ✨ Features

- 👨‍💻 Candidate management system  
- 📅 Interview scheduling  
- 🧠 Technical interview workflows  
- 📊 Feedback and evaluation system  
- 🔐 Authentication & authorization  
- ⚡ Responsive and clean UI  

---

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/ViKi10000/Intervue.git
```

2. Navigate to the project folder:

```bash
cd Intervue
```

3. Install dependencies:

```bash
npm install
cd client
npm install
```

4. Run the application:

```bash
# Start backend
npm run server

# Start frontend
cd client
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file in the server directory and add:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```
